# Quantioco.io Portal - Elite Coding BootCamp Environment 🚀

A full-stack, state-of-the-art educational portal designed for the Quantioco Engineering Fleet. This platform provides Admins with powerful cohort management tools, and Students with a gamified, beautiful learning experience.

## ✨ Features
- **Student Dashboard:** Real-time XP tracking, GitHub-style Activity Heatmap, Codeforces/Discord integration.
- **Admin Command Center:** Manage missions, metrics, and student profiles.
- **OTP Security:** Global verification for all new Cadets via Email & SMS.
- **OTP Login:** Passwordless secure entry protocol.
- **Live Classes:** Integrated 1-on-1 and cohort video sessions (Jitsi/Google Meet).
- **Assignments:** Manual deployment and module-based linking.

## 🛠 Tech Stack
- **Frontend:** React, Vite, Tailwind CSS, Framer Motion.
- **Backend:** Node.js, Express, MongoDB, JWT.
- **Payment & Enrollment:** UPI/Razorpay manual verification flow.

## 🚀 Deployment
For a production-ready environment, refer to the [Deployment Guide](https://github.com/ravi912066-netizen/quantioco-portal/blob/master/deployment_guide.md).

### Quick Local Start
1. **Repository:** `git clone https://github.com/ravi912066-netizen/quantioco-portal.git`
2. **Backend:** `cd server && npm install && node index.js`
3. **Frontend:** `cd client && npm install && npm run dev`
